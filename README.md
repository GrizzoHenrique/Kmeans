# Exercício referente ao módulo 30 do curso de ciência de dados da EBAC
**Este projeto tem como finalidade aplicar o algoritimo de clusterização K-Means**

## Etapas do projeto
1. Carregamos a base de dados diretamente do Seaborn
2. Verificar se existem dados nulos ou faltantes e realizar o tratamento
3. Realizar a exclusão das variáveis categoricas
4. Visualizar uma análise descritiva utilizando o pairplot
5. Padronizar os dados
6. Aplicar o algoritimo de K-Means com o numero de clusters igual a 3
7. Plotar duas matrizes de disperssão e realizar as análises
